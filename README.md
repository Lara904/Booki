# 🌍 Booki – Agence de voyage

Bienvenue sur le projet **Booki**, une plateforme de réservation fictive qui permet aux utilisateurs de trouver des hébergements et des activités à Marseille.

Ce projet a été réalisé dans le cadre de la formation OpenClassrooms – Développeur Web.

## 🛠️ Stack utilisée

- 💻 **HTML5** – Structure sémantique
- 🎨 **CSS3** – Design responsive, animations CSS
- 🎭 **Animations CSS**
- 📱 **Responsive Design** – Mobile-first
- 🧠 **Accessibilité** – Contrastes, navigation clavier, `alt`, `label`, `:focus`
- 🔍 **SEO de base** – `title`, `meta description`, structure optimisée

---

## ✨ Fonctionnalités

- Animation de survol sur la navigation (`:hover`, `transform`)
- Animation dynamique sur les titres via `@keyframes` lors d’un clic vers une ancre
- Cartes déroulantes au survol : texte caché qui se déploie
- Design 100% responsive (grille adaptative, media queries)
- Formulaire de contact accessible et stylé
- Utilisation de **variables CSS** pour un thème centralisé
